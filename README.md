# Legko — Современный лендинг для поиска психологов (Next.js)

Полностью обновленный современный лендинг для сервиса поиска психологов Legko. Теперь реализован на **Next.js + React + TypeScript** для максимальной производительности, масштабируемости и удобства поддержки. Проект создан на основе оригинального сайта с фокусом на максимальную конверсию и пользовательский опыт.

## 🚀 Быстрый старт

1. **Клонируйте репозиторий:**

   ```bash
   git clone <repository-url>
   cd legko-next
   ```

2. **Установите зависимости:**

   ```bash
   npm install
   ```

3. **Запустите проект в режиме разработки:**

   ```bash
   npm run dev
   # Откройте http://localhost:3000
   ```

4. **Сборка для продакшена:**

   ```bash
   npm run build
   npm start
   ```

## 🔧 Кастомизация

### Изменение цветов

Отредактируйте CSS переменные в файле `src/styles/style.css`:

```css
:root {
    --primary-color: #8263e8;    /* Основной цвет */
    --secondary-color: #667eea;   /* Вторичный цвет */
    --accent-color: #f093fb;      /* Акцентный цвет */
}
```

### Добавление новых секций

1. Добавьте React-компонент в `src/components/`
2. Импортируйте и используйте его в нужной странице (`src/pages/index.tsx`)
3. Добавьте стили в `src/styles/style.css`
4. При необходимости добавьте JS в `public/js/legko-main.js`

### Изменение контента

Весь контент теперь структурирован по React-компонентам и легко редактируется в соответствующих файлах.
