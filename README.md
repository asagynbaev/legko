# Legko - Продающий лендинг для поиска психологов

Полностью обновленный современный лендинг для сервиса поиска психологов Legko. Создан на основе оригинального сайта с фокусом на максимальную конверсию и пользовательский опыт.

## 🎨 Дизайн и особенности

- **Современный продающий дизайн** с фиолетовой палитрой (#8263e8)
- **Floating элементы** и продвинутые анимации
- **Градиентные фоны** и современные эффекты
- **Полностью адаптивная верстка** для всех устройств
- **Высокая конверсия** благодаря UX/UI решениям
- **SEO-оптимизированная структура**

## 🚀 Основные секции

1. **Hero-секция** - Обновленный УТП с социальными доказательствами

## 🚀 Быстрый старт

1. **Клонируйте репозиторий:**
   ```bash
   git clone <repository-url>
   cd legko
   ```

2. **Откройте index.html в браузере:**
   - Просто откройте файл `index.html` в любом современном браузере
   - Или используйте локальный сервер:

3. **Используя Python:**
   ```bash
   python -m http.server 8000

   ```

4. **Используя Node.js (npx):**
   ```bash
   npx serve .
   ```

5. **Используя VS Code Live Server:**
   - Установите расширение "Live Server"
   - Правый клик на `index.html` → "Open with Live Server"

## 🔧 Кастомизация

### Изменение цветов
Отредактируйте CSS переменные в файле `css/style.css`:

```css
:root {
    --primary-color: #8263e8;    /* Основной цвет */
    --secondary-color: #667eea;   /* Вторичный цвет */
    --accent-color: #f093fb;      /* Акцентный цвет */
}
```

### Добавление новых секций
1. Добавьте HTML разметку в `index.html`
2. Добавьте стили в `css/style.css`  
3. При необходимости добавьте JavaScript в `js/script.js`

### Изменение контента
Весь контент находится в `index.html` и легко редактируется.
